The application is an interface for providing review on employees and feedbacks for the review.

# Running the application

To run in App:

```shell
npm start
```


To run in dev mode:

```shell
npm run dev
```

# Tech stack involved
- React
- Redux
- Material-UI
- Nodejs
- Express
- Mongodb
- Mongoose
- Typescript

# Login credentials:
 - Admin - secret
 - employee - secret

P.S: App uses `mongodb-memory-server` for instant run and has a very short lifetime.
